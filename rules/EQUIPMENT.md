# MONDAS Equipment Catalog

## In Two Minutes

Equipment is what you carry into the field. It defines your tactical identity before the first roll. Every weapon has a dice category, properties, and 1-3 weapon-specific gambits. Every piece of gear enables a gear gambit or provides a passive benefit. Thaumatech requires a positive operator. Mundane gear just works.

Design principles, generation rules, and all tests live in `rules/DESIGN-EQUIPMENT.md`. Read that before creating entries here.

## Quick Reference

### Weapon Dice

| Category | Dice |
|-|-|
| Unarmed / Improvised | 1d4 |
| Light | 1d6 |
| Medium | 1d8 |
| Heavy | 2d6 |
| Devastating | 2d8 |

### Weapon Properties

| Property | Effect |
|-|-|
| Ranged | Near/Far, Snag at Close |
| Sidearm | Close/Near, Snag at Far |
| Thrown | Near, lost |
| Long | Close+Near, Boon at reach |
| Loud | Ticks clocks |
| Brutal | Crit = two dice |
| Subtle | Concealable |
| Slow | No Move+attack |

### Armor Values

| Value | Level | Trade-off |
|-|-|-|
| 0 | None | - |
| 1 | Light | Minimal |
| 2 | Moderate | Conspicuous, possible Sharp Snag |
| 3 | Heavy | Very conspicuous, Sharp and/or Nerve Snag |

### Legality

| Tier | Risk |
|-|-|
| Legal | None |
| Licensed | Fine if unlicensed |
| Restricted | Arrest |
| Prohibited | Federal prison |
| Black Market | Unpredictable |
